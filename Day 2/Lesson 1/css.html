Знакомство с CSS в Intocode Coding Bootcamp

CSS — язык каскадных стилей, который задаёт визуальное оформление для HTML, SVG и других языков разметки.

Преимущества CSS

  • Можно оформить что угодно и как угодно
  • Можно один раз написать стили для всех страниц сайта
  • Позволяет делать анимации
  • Легок в изучении
